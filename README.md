# Simple Python Practice Questions
## Part - 1
1. Write a program to get two numbers, multiply them, and display the result.
2. Write a program to find the perimeter of a rectangle. Formula: perimeter = 2 × (length + width).
3. Write a program to get an amount from the user and add 18% GST to it. Display the total amount including GST.
4. Write a program to get the number of apples and price per apple. Display the total cost.
5. Write a program to get the temperature in Celsius and convert it to Fahrenheit. Formula: F = (C × 9/5) + 32.
6. Write a program to get the total days from the user and convert them into years, weeks, and days.
7. Write a program to get the total seconds from the user and convert them into hours, minutes, and seconds.
8. Write a program to get the length of a square and find its area. Formula: area = side × side.
9. Write a program to get the marks of 3 subjects and find the percentage. Display the percentage.
10. Write a program to get the cost of a pen and notebook. Find and display the total cost.
11. Write a program to get the radius of a sphere and find its volume. Formula: V = (4/3) × 3.14 × r3.
12. Write a program to convert kilometers per hour (km/h) into meters per second (m/s).
13. Write a program to get any amount and calculate the 10% discount. Display the discounted price.
14. Write a program to get the base and height of a triangle and find its area. Formula: area = (base × height) / 2.
15. Write a program to get the number of students in a class and number of apples. Find how many apples each student gets and how many are left.
16. Ms. Priya from CSE wants to store her three favorite foods in one variable and display them. Write a program to help Ms. Priya.
17. Mr. Rahul from EEE wants to keep the name of his favorite book in one variable and the author’s name in another variable, then display both. Write a program to help Mr. Rahul.
18. Ms. Anjali from IT wants to record the title of her favorite song and a short description of it in a single variable. Write a program to help Ms. Anjali.
19. Mr. Karthik from MECH has three dream travel destinations and wants to store them in one variable and display them. Write a program to help Mr. Karthik.
20. Ms. Neha from ECE wants to store her role model’s name in one variable and a one‐line reason for choosing them in another variable, then display both. Write a program to help Ms. Neha.

## Part - 2
1. A student enters their exam score. If the score is 35 or above, display Pass, otherwise Fail.
2. A shopkeeper offers a discount: if a customer’s bill is above ₹500 display 10% discount applied, otherwise No discount.
3. A user enters a number. If it is even, display Even number, otherwise Odd number.
4. A person enters age. If age is greater than or equal to 18, display Eligible for voting, else display Not eligible.
5. A company offers bonus: If an employee worked more than 5 years, display bonus 5000, otherwise bonus 2000.
6. A student enters marks in three subjects. If all subjects are greater than or equal to 35, display You passed, otherwise display You failed.
7. A user enters temperature in Celsius. If temperature is below 20, display Cold, if between 20–30 display Warm, else display Hot.
8. A person types a character. If it is a vowel, display Vowel, otherwise display Consonant.
9. A bank asks to enter account balance. If balance is less than 1000, display Low balance warning, else display Sufficient balance.
10. For a given mobile number, check: if the last digit is divisible by 3, display Lucky number, otherwise display Unlucky number.
11. A bakery worker enters production quantity. If more than 100 items are baked, display Good production, else display Low production.
12. A game player enters score. If score ≥ 100, display Level Up, else display Keep playing.
13. A railway passenger enters age and gender. If age ≥ 60 and gender is Male, or age ≥ 58 and gender is Female, display Eligible for senior citizen concession, else display Not eligible.
14. A driver enters vehicle speed. If speed < 40, display Too slow, if 40–80 display Normal driving, else display Overspeeding.
15. A student enters roll number. If roll number is divisible by 2, assign Lab Group A, else assign Lab Group B.
16. A shopkeeper enters quantity of milk purchased. If quantity is greater than 5 liters, give Free coupon, else display Thank you for purchase.
17. A cricket player enters runs scored. If runs < 50 display Average, 50–100 display Good, above 100 display Excellent.
18. An ATM user enters PIN. If the entered PIN matches stored PIN, display Access Granted, else Access Denied.
19. A person enters a year. If year is divisible by 4 (and not divisible by 100 unless divisible by 400), display Leap Year, else display Not a Leap Year.
20. A student enters marks in 5 subjects. If any subject marks < 35, display Fail. Otherwise, calculate average and display result as: 35–49: Third Division, 50-59: Second Division, 60–74: First Division, ≥75: Distinction.
